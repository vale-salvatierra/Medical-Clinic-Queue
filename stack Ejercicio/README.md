
# Tarea: Simulador de Navegación con Pila

## Objetivo

Practicar la implementación de una estructura de datos tipo pila (stack) usando JavaScript.

## Instrucciones

Crea una clase llamada Stack que permita simular el historial de navegación de un navegador. Debe permitir:
Ir a una nueva página (push)
Regresar a la página anterior (pop)
Ver la página actual (peek)
Ver si el historial está vacío (isEmpty)
Ver todo el historial (print)

## Requisitos mínimos

Implementa la clase Stack con métodos apropiados.
Crea una simulación en donde:
Se visitan 3 páginas.
Se vuelve atrás una vez.
Se imprime el historial completo.
